# Snipit

A lightweight macOS screenshot tool with annotation support.

## Features

- **Global hotkey** — trigger a screenshot capture from anywhere
- **Region selection** — drag to select any area of the screen
- **Annotation tools** — draw on captures with pencil, highlighter, and text
- **Wide-gamut color** — correct rendering on P3 displays
- **macOS 14+** — built with native frameworks

## Requirements

- macOS 14.0 or later
- Screen Recording permission (prompted on first launch)

## Building

```sh
xcodegen generate
open Snipit.xcodeproj
```

Then build and run in Xcode.
